# Project Name 
> LIMINAL is a web-based code editor designed to streamline the process of organizing, editing, and managing code snippets, files, and folders for developers. This platform offers centralized code snippet management, real-time previews, and efficient file organization.

## Technologies Used
1. **Node.js:** Used for Building the backend server and handling server-side logic. 
2. **Express.js:** Simplifies the process of creating robust APIs and handling routing, middleware, and HTTP requests. 
3. **MongoDB:** Used for storing and managing data such as folders, files, and user information in LIMINAL. 
4. **React:** Used for creating the frontend of LIMINAL, providing an interactive and responsive UI for users to manage their code snippets, folders and files. 

## Features
- **Folder and File Management:** Organize code snippets into nested folders, create, update, delete files, and manage your entire project structure with ease.
- **HTML/CSS Editor:** Store and edit lengthy HTML and CSS content directly within the application.
- **Real-Time Preview:** Instantly preview your HTML and CSS changes as you code.

## Installation 
1. **Clone the repository:** <br>
`git clone https://github.com/ePortfolios/wdv349-MansonKelly-FS`<br>
`cd dev`<br>
`cd app`

2. **Install dependencies:** <br>
`npm install`

3. **Set up environment variables:** Create a `.env` file in the root of your project and configure the following variables: <br>
`PORT = 3000` <br>
`MONGODB_URL=mongodb://localhost:27017/liminal`

4. **Run the application:** <br>
`npm start`
